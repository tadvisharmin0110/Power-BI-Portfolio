# 🌍 Global Economic Analysis Dashboard using Power BI

## 📌 Project Overview
This project analyzes global economic indicators such as GDP per capita, population, and economic growth trends using Power BI. Interactive dashboards and storytelling techniques were used to generate meaningful insights from the dataset.

## 🛠 Tools Used
* Power BI
* Power Query
* DAX
* Data Modeling

## 📊 Features
* KPI Cards
* Trend Analysis
* Geographic Analysis
* Scatter Plot & Comparison Charts
* Slicers & Filters
* Bookmarks & Navigation Buttons
* What-if Parameter Analysis
* Interactive Storyboard

## 📅 Time Intelligence Functions
* YTD
* MTD
* Previous Year
* YoY Growth
* SAMEPERIODLASTYEAR

## 🌍 SDG Mapping
This project aligns with **SDG 8: Decent Work and Economic Growth** by analyzing global economic development and growth patterns.

(https://www.kaggle.com/datasets/belbino/life-expectancy-healthcare-spending-1960-present)
